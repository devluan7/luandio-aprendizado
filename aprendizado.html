História da WEB

Início da Página

Em 1994 foi onde surgiu o "Mark I", básicamente uma máquina de calculo, é considerado o primeirocomputador criado, entretanto, a ideia do "computador", já tinha surgida bem antes, com Alan Turing e com outras maquinas de cálculo, porém, o Mark I conseguia gerar calculos bem mais completos. Logo após seu surgimento, foram sendo criados outros computadores, como o "Colossus" em 1946(Meados da 2° Guerra Mundial), feito pela Alemanha para cripto análise, jutamente do Eniac, criado pelos EUA, que já começou a ser um computador um pouco mais automático. Nessa mesma época foi criado o termo "bug", pois havia insetos que entravam no computador e acabavam comprometendo seu sistema, então, quando isso acontecia, diziam que tinham dado "bug", que é como é chamado inseto em inglês.

Com o passar dos anos, a tecnologia foi progredindo bastante, e com isso foram criados os computadores de segunda geração (1959 - 1965), que utilizavam transistores, e assim, diminuindo o circuito do computador fazendo com que agora fosse menor. Logo após isso, veio a terceira geração (1965 - 1970) com circuitos integrados e com computadores menores ainda, e finalmente a quarta geração, que é a que usamos até os dias atuais.

Como surgiu

Mas afinal, como surgiu a internet? Bom, o primeiro sistema de comunicação foi o telégrafo, com codigo morse em 1958, nessa época, Joseph Licklider teorizou que pudesse existir uma rede que todos poderiam teorizar através dela, desde então, muita coisa aconteceu, como a Arpanet, que foi um sistema de comunicação em pacotes, sua primeira conexão foi feita em 1969, onde tentaram mandar uma mensagem escrita "Login", a comunicação foi feita de uma Universidade da california para o Instituto de pesquisa de Stanford. Porém, nessa primeira tentativa de comunicação, acabou tendo uma queda de conexão, e apenas as letras "Lo" chegaram no Instituto. Desde então a comunicação foi evoluindo cada vez mais, com a Milnet, nfsnet, entre outras.

O que são Clients

Início da Página

O client é básicamente quando você consome a informação de algum lugar, ou seja, o client é o dispositivo que está consumindo uma informações de um servidor, diretamente da internet. Quando você acessa um site na internet, seu computador pega as informações que o de um servidor te fornece, ou seja, arquivos html, css, imagens, áudios, etc. Esses arquivos são chamados de arquivos estáticos, e o navegador pega esses arquivos e armazena na memória cache, e isso faz com que, quando você entrar em um site pela segunda vez, todos arquivos do memso, seja carregado bem mais rapido, pois as informações já estão armazendas no seu computador.

Ainda na parte do client, a pessoa que faz o site, deve se atentar as diferentes plataformas que poderão se conectar no website, fazendo com que assim, tenha um design responsivo, ou seja, se uma pessoa entra pelo celular, o design da página precisa estar adaptado para um dispositivo móvel, e o mesmo se aplica para caso a pessoa entre por um computador, notebook, TVs. É necessário ter uma responsividade para diferentes navegadores (Chrome, FireFox, Opera,Edge, etc), que geralmente tem um padrão, porem, existe algumas diferenças de como determinadas tags se comportam em diferentes navegadores.

Aplicações WEB

Aplicações WEB são soluções criadas que possuem a internet como meio de comunicação entre Client x Server, não sendo necessárias a sua instalação. Ou seja, o desenvolvedor front na parte dos navegadores por exemplo, é diferente do dev front na parte de aplicativos.

O que são Servers

Início da Página

Servidores são em sua grande maioria, computadores que armazena toda informação que um site, empresa ou aplicativo possui, com isso, o server faz a conexão com a internet, ou rede local. O servidor tende a ser um computador mais potente, pois ele terá uma conexão com diversos dispositivos, e esse tráfego de informações exige um bom processamento. Os servidores podem ser tanto computadores "simples", quanto podem ser enormes servidores, que são chamados "Data centers", onde é armazenado varios computadores, que são utilizados para atender diversos clients, sdão usados por grandes empresas.
Existem diversos tipos de servidores, tais quais:

Servidores de arquivos, que é um computador onde voce vai direcionar todos os arquivos, imagens, videos, documentos, etc
Servidores de Streaming, servidores dedicados a fazer streaming de dados, geralmente de vídeos
Servidores WEB, que é um servidor que vai disponibilizar os arquivos de uma pagina web, arquivos html, arquivos de foto, css. (Apache, Nginx).

Sobre Linguagens de programação WEB

Início da Página

Linguagem de programação é uma linguagem escrita e formalque especifica um conjunto de instruções e regras usadas para gerar programas. Segundo a ao RedMonk as 5 Linguagens mais utilizadas atualmente, é, JavaScript, Python, Java, PHP e C#. É importante lembrar que HTML NÃO É linguagem de programação e sim um arquivo de marcação.

Server Side e Client Side

O lado do servidor tem algumas lingugens especificas que rodam nele, ex: php, ruby, javascript, etc. Quando a você criar uma aplicação usando essas linguagens no servidor, ele vai rodar somente do lado server, e não no computador do client.

Já o lado do client (O lado que vai acessar o site), basicamente a linguagem que mais vai ser executada desse lado, vai ser o javascript (Tambem o Html e CSS, porem, como dito antes, não são linguagens de programação). A linguagem do javascript roda em ambos os lados pois o navegador possui um mecanismo(v8.dev) para rodar o JS, porem ele não faz isso sozinho, ele tem o auxilio do NodeJS que é como se fosse um navegador que roda o V8, e esse Node pode ser instalado no servidor.

Introdução ao HTML

Início da Página

A estrutura do HTML é gerado básicamentes por tags, e o HTML consiste em duas partes, no head, que é a cabeça do arquivo e o body, que é o corpo do arquivo. Tudo que tiver entre o body, é o que aparece na página para o usuário (basicamente essa página inteira). Já o Head, são informações que ele vai carregar, antes de mostrar para o usuário, por exemplo o title, que é o titulo da página.
Cada tag tem sua peculiaridade, geralmente elas tem o padrao de < > e com algo escrito dentro (< p >), e para fechar < / > ( < / p >). Tem também a tag input, que ela pode criar um campo que pode ser formatado como texto, número, botão, imagem etc.

Exemplos de TAGS


< b > < \ b > Essa tag faz com que um texto fique em Negrito
< p > < \ p > É a tag de parágrafo, essa pagina inteira, foi dividida em parágrafos
< br > < \ br > Dentro de um Parágrafo, essa tag faz com que pule uma linha (Assim como nessa explicação).
< ol (ul) > < \ ol (ul) > Ordered List e Unordered List, são tags utilizadas para se fazer uma lista dentro de um fomulário, a lista ordenada pode ordenar a lista com "1,2,3", "I,II,III", "ABC" e até mesmo de tras para frente. Já a não ordenada, coloca alguns simbolos na lista, como bolinha, bolinha vazia, etc. Essa tag vem acompanhada da tag < li > < \ li > que o que da a estrutura de uma lista.
< a href > < \ a > Essa tag básicamente linka algo a um texto, imagem ou objeto, podendo ser um link de outra página, um arquivo, ou até mesmo te mandar para outra parte de uma mesma página.
Existem inúmeras tags, como as Hs, title, etc. Estou aprendendo ainda e colocando em prática cada uma delas, ao final do curso espero ter dominado grande parte delas.
